The five

A spectrum from literal to expressive — pick the personality.

A · Utilitarian

Box Office

A thermal will-call ticket: mono type, perforations, a real barcode + order number. Reads instantly as "an actual ticket."

B · Structural

Torn Half

The tear is the hero — a two-part ticket with a detachable date stub. Doubles as a tidy right-edge tab in the feed.

C · Raw

Xerox Flyer

DIY show-flyer energy: condensed caps, halftone grain, taped corners, a stamped button. Most WXYC-freeform — the bold pick.

D · Refined

Handbill

A letterpress concert handbill: centered, hairline rules, spelled-out date. The quiet, editorial end of the range.

E · Playful

Marquee

A theatrical lightbulb border with a "now playing" band. Warm and eye-catching without leaving the amber palette.

Still true for all five

  • One amber signal. Every style uses the app's accent hue (23°) so "on tour" never competes with the green on-air dot.
  • Match on a name, guarded. Triangle Shows has no artist ID — highlight only on a normalized exact match against GET /api/events?search={artist}&start={today}; treat fuzzy hits as no-match for v1.
  • State-aware. The status field (on_sale / sold_out / cancelled / free) drives the pill; only ever show date >= today, soonest first.
  • Local by nature. ~21 Raleigh–Durham–Chapel Hill venues — "playing near you" is literally true for WXYC listeners; keep the copy local, not national.
